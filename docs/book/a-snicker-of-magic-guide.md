## 2. 한 줄 요약

**단어를 볼 수 있는 소녀 Felicity가 마법이 사라진 작은 마을 Midnight Gulch에서 가족의 저주와 자신의 두려움을 마주하며, 진짜 '집'의 의미를 찾아가는 마법적 성장소설.**

---

## 3. 세계관 / 배경 구조

### 3.1 Midnight Gulch

| 요소 | 설명 |
|---|---|
| 마을 이름 | Midnight Gulch |
| 위치 | 미국 테네시주의 가상 마을 |
| 과거 | 한때 마법으로 유명했던 마을. 사람들의 재능과 감정이 현실에 마법처럼 나타났음 |
| 현재 | 강력한 마법은 사라졌고, 작은 흔적만 남아 있음 |
| 분위기 | 따뜻하지만 쓸쓸한 시골 마을, 오래된 전설과 가족사가 깔린 공간 |
| 중심 갈등 | 왜 마법이 사라졌는지, 왜 Felicity 가족은 계속 떠돌게 되었는지 밝혀가는 과정 |

### 3.2 배경의 기능

Midnight Gulch는 단순한 무대가 아니라, 인물들의 상처와 기억이 쌓인 장소다.
이 마을은 Felicity가 처음으로 "home"이라는 감각을 느끼는 공간이며, 동시에 가족의 오래된 저주와 마을의 마법 상실이 얽혀 있는 핵심 장소다.

---

## 4. 마법 시스템

이 작품의 마법은 주문, 전투, 능력 대결 중심이 아니다.
사람의 재능, 기억, 감정, 상처가 현실에 살짝 드러나는 **마법적 리얼리즘**에 가깝다.

| 마법/능력 | 보유자 | 설명 | 서사적 기능 |
|---|---|---|---|
| 단어 보기 / 단어 수집 | Felicity Pickle | 사람, 장소, 사물 주변에 떠오르는 단어를 볼 수 있음 | 타인의 감정과 숨겨진 진실을 읽는 장치 |
| Know-how | Jonah Pickett | 누군가를 어떻게 도와야 할지 직감적으로 아는 능력 | Felicity가 마을과 연결되도록 돕는 장치 |
| 음악적 마법 | Stone & Berry Weatherly | 기타와 밴조 음악을 통해 마법적 영향력을 가짐 | 마을의 전설과 저주의 원인 |
| 기억을 건드리는 음식 | 마을의 음식/상점 | 특정 음식이 기억과 감정을 불러일으킴 | 마을의 마법적 분위기 강화 |
| 저주 | Weatherly 가문 / Pickle-Harness 가족과 연결 | 방황, 실패, 관계 단절 등으로 이어짐 | 가족사와 현재 갈등을 연결 |

## 5. 주요 등장인물

## 5.1 Felicity Juniper Pickle

| 항목 | 내용 |
|---|---|
| 역할 | 주인공, 1인칭 화자 |
| 나이 | 12세 |
| 별명 | Flea |
| 특징 | 단어를 볼 수 있는 word collector |
| 성격 | 섬세함, 상상력 풍부함, 소심하지만 내면은 강함 |
| 결핍 | 계속 이사 다니는 삶 때문에 '집'에 대한 갈망이 큼 |
| 욕망 | Midnight Gulch를 진짜 집으로 만들고 싶어 함 |
| 성장축 | 능력을 숨기는 아이 → 자기 목소리와 재능을 받아들이는 아이 |

### 캐릭터 해석

Felicity는 타인의 감정과 분위기를 예민하게 감지하는 아이다.
그 능력이 작품 안에서는 "단어를 보는 능력"으로 표현된다.
그녀의 가장 큰 욕망은 특별한 마법을 쓰는 것이 아니라, 더 이상 떠돌지 않고 한곳에 머무르는 것이다.

## 10. 캐릭터 IP 분석 관점

### Felicity의 IP 포인트

- 공중에 떠다니는 단어를 보는 시각적 능력
- "집을 찾는 아이"라는 보편적인 감정선
- 소심하지만 섬세한 주인공성
- 영어 학습 콘텐츠와 연결하기 좋은 단어 수집 설정

### Jonah의 IP 포인트

- 누군가를 돕는 방법을 아는 능력
- 밝고 다정한 조력자 포지션
- 휠체어 사용 캐릭터지만, 장애 자체보다 인물의 능동성과 관계성이 강조됨

### Midnight Gulch의 IP 포인트

- 마법이 완전히 사라진 것이 아니라, 흔적으로 남아 있는 마을
- 음식, 음악, 단어, 기억 같은 일상적 소재를 마법으로 확장 가능
- 에피소드형 콘텐츠로 만들기 좋음

## 13. 최종 요약

*A Snicker of Magic*은 겉으로는 마법이 사라진 마을의 비밀을 풀어가는 이야기지만, 실제 핵심은 **상처 입은 가족과 공동체가 다시 연결되는 과정**이다.

Felicity의 단어를 보는 능력은 단순한 판타지 설정이 아니라, 사람의 감정과 진실을 읽는 은유다.
이 작품에서 마법은 거대한 힘이 아니라, 누군가를 이해하고, 기억하고, 붙잡아주는 작은 순간들 속에 있다.
